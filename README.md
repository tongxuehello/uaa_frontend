# 说明

基于reactjs搭建的前端模板库

运行项目需要：

1. 安装node
2. 升级npm

# 运行方法

1. npm install
2. webpack-dev-server --config webpack.config.js --colors --progress --port 8081
3. http://localhost:8080/webpack-dev-server/
